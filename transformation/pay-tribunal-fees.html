Scope of exemplar

In line with new MoJ policy, employment tribunals will start charging fees from 29 July. The ICT to implement this ministerial priority includes a payment mechanism, which has been improved through digital input on user testing and the Digital by Default Standard. Phase 2 runs from July to October and will improve the transaction further by including remission claims.

For more information about this project please contact us via our feedback form.
